  WARBAND NEXUS - Version 2.2.0 (23.02.2026)
═══════════════════════════════════════════════════════════════════════════

NEW FEATURES
────────────
• Gold Target: Smart automatic gold management system
  - Set target gold amount per character (default: 10g)
  - Three modes: Deposit Only, Withdraw Only, or Both
  - Fully automatic deposits AND withdrawals when bank opens
  - Uses C_Bank API (DepositMoney/WithdrawMoney)
  - Smart 2-second cooldown between operations
  - Silent operation (no spam)
  - Integrated into Items tab header
  - Custom themed radio button widget
  - 11 language translations included

• Personal Items Tab Restructured
  - 4 separate sub-tabs: Inventory, Personal Bank, Warband Bank, Guild Bank
  - Each sub-tab shows items from its specific source
  - Guild Bank tab automatically enabled/disabled based on guild membership
  - Gold display per sub-tab (Inventory, Warband Bank, Guild Bank)
  - Character-guild tracking with automatic guild change detection
  - Guild membership changes detected on login (kick/leave/join)
  - Guild bank data persists per guild name (shared across characters in same guild)

• Storage Tab Enhanced
  - Guild Bank section added to hierarchical storage browser
  - Multiple guild banks displayed separately if characters are in different guilds
  - Guild Bank items organized by type (same as Warband and Personal)
  - Search functionality includes Guild Bank items
  - Section ordering: Personal Banks → Warband Bank → Guild Bank(s)
  - Each guild bank displays its guild name in the header

• Event-Driven Guild Bank System
  - Real-time automatic re-scan when guild bank contents change
  - GUILDBANKBAGSLOTS_CHANGED event triggers smart throttled re-scan (2.5s)
  - Cache updates automatically when items/gold are deposited or withdrawn
  - All characters in the same guild see updated data immediately on next login
  - Comprehensive debug logging for troubleshooting guild bank operations

IMPROVEMENTS & CHANGES
──────────────────────
• Items Tab: Split combined "Personal Items" into dedicated Inventory and Personal Bank tabs
• Items Tab: Guild Bank now always visible (disabled if not in guild)
• Items Tab: Guild bank gold now uses cached value instead of live API (fixes cross-character display bug)
• Storage Tab: Guild Bank section integrated into storage hierarchy
• Storage Tab: Fixed expand/collapse functionality for Guild Bank headers
• Data Service: Added GetInventoryItems() and GetBankItems() functions
• Character Service: Added guildName tracking to character data
• Guild Bank Scanner: Caches guild bank gold amount during scan
• Guild Bank Scanner: Automatic re-scan on slot changes (event-driven)
• Core: Guild change detection on PLAYER_ENTERING_WORLD event
• Core: GUILDBANKBAGSLOTS_CHANGED handler with throttled re-scan
• UI: Custom radio button widget added to SharedWidgets
• UI: Auto-sizing buttons based on localized text width
• Performance: Removed debug logging for cleaner operation (except Guild Bank debugging)
• Locales: Updated translations for new sub-tabs (5 languages)

BUG FIXES
─────────
• Fixed: Guild bank gold displaying wrong character's amount when switching characters
• Fixed: Guild bank items not updating after deposits/withdrawals
• Fixed: Guild Bank header not expanding when clicked in Storage tab
• Fixed: Multiple guild banks not displaying if characters are in different guilds
